# オシイレ プライバシーポリシー

制定日：2026年6月19日
改定日：2026年6月27日

本プライバシーポリシーは、iOS アプリ「オシイレ」（以下「本アプリ」）におけるユーザー情報の取り扱いについて定めるものです。

## 1. 基本方針

本アプリには、アカウント登録や、開発者が運営するデータ同期サーバーがありません。あなたが本アプリに登録したデータ（予約・所有グッズの情報、写真、メモ等）は、すべてあなたの端末内にのみ保存され、開発者がその内容にアクセスする手段はありません。

一方で、本アプリの品質改善のために、どの画面・機能が使われているかといった**匿名の利用状況データ**を収集しています（詳細は第4項）。この利用状況データに、あなたが登録したグッズの内容・写真・メモや、氏名・住所・メールアドレスなどの個人を特定する情報は一切含まれません。

## 2. カメラ・写真へのアクセス

本アプリは、グッズの写真を登録する目的でのみ、カメラまたは写真の選択機能を使用します。

- **カメラ**：その場で撮影する際に、カメラへのアクセス許可を求めます。
- **写真**：iOS 標準の写真選択画面を用います。この方式では、ユーザーが選んだ写真のみが本アプリに渡され、写真ライブラリ全体へのアクセス許可は求めません。

いずれの場合も、取り込んだ写真はユーザーの端末内にのみ保存され、外部に送信されることはありません。

## 3. ネットワーク通信について

本アプリには、ネットワーク通信を伴う以下の機能があります。いずれもユーザーの操作によって起動します。各機能における情報の取り扱いは以下のとおりです。

### 3-1. URL取り込み機能

ユーザーが商品ページの URL を貼り付けると、商品名・価格・画像 URL 等を下書きとして取り込みます。

- この通信は、ユーザーが指定した商品ページ（外部ショップのサーバー）に対して、ユーザーの端末から直接行われます。
- アクセス先のショップにおける情報の取り扱いには、各ショップのプライバシーポリシーが適用されます。

### 3-2. バーコード取り込み機能

ユーザーが「バーコードで取り込む」を実行すると、読み取ったバーコード番号（JAN コード）を Yahoo!ショッピング 商品検索 API に送信し、商品名・価格・画像等を下書きとして取り込みます。

- 送信される情報はバーコード番号（JAN コード）のみです。氏名・住所などの個人情報は送信されません。
- Yahoo!ショッピング API のご利用には、Yahoo Japan の利用規約が適用されます。

### 3-3. 問い合わせ・フィードバック機能

本アプリの設定画面から問い合わせ・改善要望を送る際、外部サービスである Google フォームをブラウザで開きます。この際、不具合の調査に役立てる目的で、以下の端末情報があらかじめ入力されます。

- あらかじめ入力される情報は、アプリのバージョン・ビルド番号・iOS のバージョン・端末の機種識別子（例：iPhone17,2）です。氏名・住所・連絡先などの個人情報は含まれません。
- フォームおよびそこに入力・送信された内容の取り扱いには、Google のプライバシーポリシーが適用されます。
- ユーザーがフォームに自ら記入した内容（ご要望・連絡先等）は、開発者が問い合わせ対応の目的でのみ利用します。

## 4. アクセス解析（利用状況データ）について

本アプリは、機能改善のために Google が提供する **Firebase Analytics** を利用し、匿名の利用状況データを収集します。

### 収集の目的

どの画面や機能が使われているか、どこで利用をやめてしまうか等を統計的に把握し、本アプリの改善に役立てます。

### 収集する内容

- 画面の表示や機能の利用といった操作イベント（例：グッズの登録、URL 取り込み、到着の記録）
- アプリのバージョン、OS のバージョン、端末の種類
- おおよその地域（IP アドレスから推定されます。IP アドレス自体は保存しません）
- アプリのインストールごとに割り当てられる匿名の識別子

これらは統計的な集計のために用いられ、**個人を特定するものではありません**。あなたが登録したグッズの内容・写真・メモは送信されません。

### 広告・トラッキングについて

本アプリは、広告を目的としたアプリ横断的なトラッキング（IDFA／広告識別子の利用）を行いません。そのため、いわゆる「トラッキングの許可」を求めるダイアログ（App Tracking Transparency）は表示しません。

### データの処理者

利用状況データは Google により処理されます。Google における取り扱いについては、[Google プライバシーポリシー](https://policies.google.com/privacy) をご確認ください。

### 収集の停止について

本アプリには、利用状況データの送信を個別に停止する設定はありません。ただし、収集されるデータは上記のとおり匿名であり、個人を特定しません。本アプリを削除すると、以降の収集は行われません。

## 5. 第三者への提供

本アプリは、以下を除き、ユーザーの情報を第三者に提供しません。

- 第4項に記載した、匿名の利用状況データの Google による処理
- 第3項に記載した、ユーザーの操作によって外部サービスへ直接行われる通信

本アプリは、広告 SDK を使用していません。

## 6. データの削除

本アプリ内のデータは、アプリ内の削除操作、または本アプリの削除によって端末から削除できます。本アプリを削除すると、第4項に記載した利用状況データの収集も以降行われません。

## 7. 本ポリシーの変更

本ポリシーの内容は、必要に応じて変更されることがあります。変更後の内容は本ページにて掲示します。
