# Oshiire プライバシーポリシー

制定日：2026年6月19日

本プライバシーポリシーは、iOS アプリ「Oshiire」（以下「本アプリ」）におけるユーザー情報の取り扱いについて定めるものです。

## 1. 開発者によるデータ収集について

**本アプリは、ユーザーの情報を一切収集しません。**

本アプリにはサーバーやバックエンドがなく、アカウント登録も不要です。本アプリに登録されたデータ（予約・所有グッズの情報、写真、メモ等）は、すべてユーザーの端末内にのみ保存されます。開発者はこれらのデータにアクセスする手段を持ちません。

## 2. カメラ・写真ライブラリへのアクセス

本アプリは、グッズの写真を登録する目的でのみ、カメラおよび写真ライブラリへのアクセス許可を求めることがあります。取得した写真はユーザーの端末内にのみ保存され、外部に送信されることはありません。

## 3. ネットワーク通信（URL取り込み・バーコード取り込み機能）について

本アプリには、以下のネットワーク通信を伴う機能があります。いずれもユーザーの操作によって起動し、取得した内容は端末内でのみ処理されます。開発者がその内容を受信・保存することはありません。

### 3-1. URL取り込み機能

ユーザーが商品ページの URL を貼り付けると、商品名・価格・画像 URL 等を下書きとして取り込みます。

- この通信は、ユーザーが指定した商品ページ（外部ショップのサーバー）に対して、ユーザーの端末から直接行われます。
- アクセス先のショップにおける情報の取り扱いには、各ショップのプライバシーポリシーが適用されます。

### 3-2. バーコード取り込み機能

ユーザーが「バーコードで取り込む」を実行すると、読み取ったバーコード番号（JAN コード）を Yahoo!ショッピング 商品検索 API に送信し、商品名・価格・画像等を下書きとして取り込みます。

- 送信される情報はバーコード番号（JAN コード）のみです。氏名・住所などの個人情報は送信されません。
- Yahoo!ショッピング API のご利用には、Yahoo Japan の利用規約が適用されます。

## 4. 第三者への提供

本アプリは、ユーザーの情報を第三者に提供しません。また、広告 SDK・アクセス解析 SDK 等の第三者モジュールを使用していません。

## 5. データの削除

本アプリ内のデータは、アプリ内の削除操作、または本アプリの削除によって端末から削除できます。

## 6. 本ポリシーの変更

本ポリシーの内容は、必要に応じて変更されることがあります。変更後の内容は本ページにて掲示します。
